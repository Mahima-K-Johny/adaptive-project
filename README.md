 Project Title:Adaptive Exam & Skill-Based Course Recommendation System

 
The Adaptive Exam & Skill-Based Course Recommendation System is a modern web-based learning platform developed using the MERN stack (MongoDB, Express.js, React.js, Node.js). The system is designed to enhance student learning by providing personalized assessments and targeted study and skill-building recommendations based on individual performance and preferences. Unlike traditional online examination platforms, the system dynamically adjusts the difficulty of exam questions according to the student’s responses, simulating an adaptive learning environment without requiring complex artificial intelligence.
Students can take exams, monitor their performance, and view topic-wise analytics to identify areas of improvement. Based on these insights, the platform recommends relevant study materials such as PDFs, articles, videos, and skill-building courses. A unique feature of the system is its interactive chatbot, which engages students in conversation to understand their learning goals, current skill levels, and preferences. The chatbot, powered by React and Javascript, suggests courses or learning paths tailored to the student’s interests and needs, helping them improve specific skills and gain targeted knowledge.
Instructors can manage question banks, assignments, and track student progress, while administrators can oversee platform usage and generate analytics reports. The backend, built with Node.js and Express.js, handles authentication, exam logic, and performance tracking, while MongoDB stores user data, question banks, learning materials, and course recommendations. The frontend, developed with React.js, provides interactive dashboards for students, instructors, and administrators, ensuring a seamless user experience. JWT-based authentication and role-based access control ensure security and privacy.
Designed for scalability, usability, and maintainability, this platform bridges gaps in student performance, promotes targeted learning, and empowers students to improve skills efficiently. The integration of an intelligent, interactive chatbot makes this system a comprehensive solution for adaptive learning and skill enhancement, making it valuable for educational institutions seeking modern, personalized learning platforms.


Technologies Used

MongoDB – Stores user data, exam details, performance records, and course information.
Express.js – Handles backend logic, APIs, and routing.
React.js – Builds the interactive user interface and dashboards.
Node.js – Runs the backend server and manages authentication and exam logic.
JWT Authentication – Ensures secure login and role-based access control.
React & javascript (Chatbot) – Powers the chatbot for course and skill recommendations.
